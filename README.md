# amandaedwards.co

The official website for Amanda Edwards, independent children's book author.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Language**: TypeScript
- **Deployment**: Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Pages

- **Home** — Hero, featured books, about preview, testimonials, blog preview, newsletter
- **Books** — Full book catalog with individual detail pages
- **About** — Author bio, timeline, fun facts
- **Fun Stuff** — Free downloadable activities, coloring pages, crafts
- **Blog** — Parenting tips, behind-the-scenes, author journey
- **Contact** — Contact form with reason selection, social links, FAQ

## Deployment

This project is configured for Vercel deployment:

```bash
npm run build
```
